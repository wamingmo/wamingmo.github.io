作为一个计算机专业的学生，居然苦恼于电脑C盘和D盘爆满的情况。

虽然客观条件确实差（电脑总共只有512g，和我的手机一样大。。。。）但是也不至于屡屡爆满，尤其是在我还给它重分配过很多次的情况下。

下面就介绍两个好用的工具（目前是两个，当然也可能是三个，因为在我写这个文章的时候，有一个软件还在体验中）

WizTree

关于我怎么发现回收站也占空间这件事…

在C盘和D盘持续爆满的高压环境下，我想过换电脑，但是迫于形势，痛定思痛，决定彻查一下，然后发现了这个软件

Download WizTree — Free Disk Space Analyzer

打开之后如下：


这个是我清理过的，原来有一个无敌大的文件，叫：$RECYCLE.BIN，有100多g。。

搜了一下发现是回收站占用的空间

D:\$RECYCLE.BIN 是 D 盘的回收站文件夹。

简单来说，你在 D 盘里删除的任何文件（直接按 Delete 键删除，而不是 Shift + Delete 彻底删除），其实并没有真正从硬盘上消失，而是被系统偷偷移动到了这个隐藏的 $RECYCLE.BIN 文件夹里。

每个盘符（C 盘、D 盘、E 盘等）根目录下都会有这么一个隐藏的 $RECYCLE.BIN 文件夹。你在桌面上看到的那个“回收站”图标，实际上是一个聚合视图，它把所有盘的 $RECYCLE.BIN 里的内容统一展示给了你。

🔍 为什么这个文件夹会占用很大空间？

既然你现在电脑空间紧张，这个文件夹很可能塞满了你以前删除的开发项目、大视频或者压缩包。

因为它是隐藏的系统文件夹，正常情况下 Windows 会把它藏起来。如果你用 WizTree、SpaceSniffer 等空间分析工具扫描，就会发现它以 $RECYCLE.BIN 的名字暴露了出来，并且可能占用了几十个 G 的空间。

🛠️ 怎么安全地清理它？

绝对不要直接右键删除 $RECYCLE.BIN 文件夹本身！ 它是系统组件，强行破坏可能导致回收站机制出错。

最安全、最正规的清理方式有以下几种：

方法 1：清空桌面回收站（最简单）

直接去桌面上右键点击 “回收站” 图标 -> 选择 “清空回收站”。这会同时清空包括 C 盘、D 盘在内的所有盘的暂存文件。

方法 2：使用属性限制 D 盘回收站的最大容量

如果不想让 D 盘的回收站无限制地吃空间，你可以限制它的大小：

  1. 在桌面上右键点击 “回收站”，选择 “属性”。
  2. 在弹出的窗口中，选中 D 盘。
  3. 勾选 “自定义大小”，给它设置一个上限（例如 4096 MB，即 4GB）。这样一旦超过 4GB，它会自动永久删除最旧的文件，不再继续吃空间。
  4. 或者，如果你非常有把握，可以勾选 “不将文件移到回收站。移除文件后立即将其永久删除”（不过这样以后删错东西就很难找回了，需谨慎）。

💡 名字里为什么有 $S-1-5-21-… 这样的子文件夹？

如果你点进 $RECYCLE.BIN，会看到里面有长串数字命名的文件夹。这些是 Windows 的用户安全标识符（SID）。因为 Windows 支持多账号登录，系统用这些一长串的数字来区分“哪个文件是哪个用户删的”，确保 A 用户进回收站看不到 B 用户删掉的隐私文件。

这下只需要清理回收站，就省了很多空间出来

我又活过来了。
